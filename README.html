使用 gulp 参与编辑

  1. Fork 此项目
  2. 使用 Github Clone 到本地
  3. 打开命令行/终端
  4. 跳转至对应目录
  5. 命令行输入 npm install
  6. 输入 gulp 回车
  7. 开始监控 /post/**/*.md 文件修改，当 *.md 文件被修改时，将自动同步 对应的 /**/*.html 文件。

修改完成后使用 github 发送 pull requests

HTML 特殊处理

因为官方英文文档中夹杂着一些带 class 的 <div> 所以通过 replace 做了一个替换。

<!---<div class="bullet">-->

Handlebars 模板看起来很像 HTML ，Handlebars 表达式嵌入在 HTML 中。

html = html.replace(/<!---([^-]+)-->/g,'$1')

可查看 /posts/index.md 的源码以理解 <!---<tag>--> 的用法。
